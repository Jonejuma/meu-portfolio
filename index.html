Olá — sou Jone Juma Amade, criativo moçambicano natural de Nampula, a trabalhar em Maputo. Combino design gráfico e desenvolvimento web para criar soluções digitais que realmente funcionam.

Formado em Suporte Informático, tornei-me autodidata em design e web — trabalho com Adobe Illustrator e Photoshop para criar logos, identidades visuais, flyers, banners e conteúdo para redes sociais, e com HTML, CSS e JavaScript para construir sites modernos e responsivos.

Para mim, cada projeto é uma oportunidade de impressionar. Do primeiro briefing ao resultado final — entrego com atenção a cada detalhe.
